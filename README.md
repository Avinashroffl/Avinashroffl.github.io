# Avinash R — macOS Portfolio

A macOS Finder-inspired personal portfolio website built with vanilla HTML, CSS and JavaScript.

## Live

**[avinashroffl.github.io](https://avinashroffl.github.io/)**

## Features

- **Lock Screen** — Click or press any key to unlock, just like macOS
- **Finder Window** — Single window with a sidebar and tab-based content (About, Experience, Projects, Education)
- **Sidebar Navigation** — Favorites and external links (GitHub, LinkedIn, Contact)
- **Back / Forward** — Browser-style history navigation within tabs
- **Apple Menu** — Top-left  icon with "About This Mac" and "Lock Screen" dropdown
- **Dock** — Bottom dock with animated hover effects to switch between sections
- **Draggable Window** — Click and drag the title bar to reposition the Finder window
- **Responsive** — Mobile-friendly with a collapsible sidebar (hamburger menu) and adaptive layout

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, backdrop-filter, animations)
- Vanilla JavaScript (no frameworks or libraries)
- Google Fonts (Inter)
- Inline SVG icons (Lucide-style)

## Structure

```
├── index.html        # Main HTML
├── styles.css        # All styles + responsive breakpoints
├── script.js         # Tab switching, drag, lock screen, navigation
├── profile.jpeg      # Profile photo
├── background.png    # Desktop wallpaper
└── README.md
```
